# Katurro

## Descripción

Katurro es un sitio web dedicado a dar a conocer el fascinante mundo del café y la rica cultura del Eje Cafetero. A través de páginas informativas, galerías visuales y actividades interactivas, los usuarios pueden explorar la historia, el proceso de producción, las tradiciones y la importancia cultural del café en Colombia.

## Características

- **Historia del Café**: Aprende sobre los orígenes y la evolución del café.
- **Proceso de Producción**: Descubre cómo se cultiva, cosecha y tuesta el café.
- **Galería**: Explora imágenes del cultivo y la cultura cafetera.
- **Mapas Interactivos**: Navega por el Eje Cafetero con mapas dinámicos.
- **Actividades y Juegos**: Participa en actividades relacionadas con el café, como catas y mapas del tesoro.
- **Información de Contacto**: Encuentra formas de contactar y apoyar el proyecto.

## Tecnologías Utilizadas

- HTML5 para la estructura del sitio
- CSS3 para el diseño y estilos
- JavaScript para interactividad y funcionalidades dinámicas

## Cómo Usar

1. Clona o descarga el repositorio.
2. Abre el archivo `index.html` en tu navegador web preferido.
3. Navega por las diferentes secciones usando el menú de navegación.

## Estructura del Proyecto

- `index.html`: Página principal del sitio
- `historia.html`: Información histórica sobre el café
- `proceso.html`: Detalles del proceso de producción del café
- `galeria.html`: Galería de imágenes
- `mapa.html`: Mapa del Eje Cafetero
- `mapa-tesoro.html`: Juego de mapa del tesoro
- `cata-cafetera.html`: Información sobre catas de café
- `actividades.html`: Actividades interactivas
- `apoyo.html`: Información de apoyo al proyecto
- `caracteristicas.html`: Características del café
- `contacto.html`: Página de contacto
- `formulario.html`: Formulario de contacto
- `sobre-nosotros.html`: Acerca del proyecto
- `css/styles.css`: Archivo de estilos CSS
- `js/`: Carpeta con archivos JavaScript (navbar.js, mapa.js, etc.)
- `images/`: Carpeta con imágenes del sitio
